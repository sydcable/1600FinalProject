# Tools for Working with Bitmap and Vector Images

There are many software applications designed for creating, editing, and exporting bitmap and vector graphics. Here are some popular options:

## Bitmap Image Tools

- Adobe Photoshop: Industry-standard for photo editing and raster graphics.
- Corel PHOTO-PAINT: Professional bitmap editing tool.
- Paint.NET: Lightweight and beginner-friendly image editor.

## Vector Image Tools

- Adobe Illustrator:  Leading software for creating vector graphics.
- Inkscape: Free and open-source vector graphics editor.
- CorelDRAW: Professional vector design tool for illustrations and layouts.
- Affinity Designer: Affordable alternative for vector and raster design.
